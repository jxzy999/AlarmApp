# ⏰ 小闹钟

## ✨ 核心功能 | Features

- **📅 智能节假日模式**
  
  - 自动识别法定节假日和调休补班（基于 HolidayService）。
  
  - “上班族”福音：假期自动静音，调休自动响铃。
  
  - 来源：[NateScarlet/holiday-cn](https://github.com/NateScarlet/holiday-cn)（每年国务院文件发布后更新）
  
  - 使用仓库公开 JSON（字段包含 `name`/`date`/`isOffDay`），将 `isOffDay=true` 视为休息日，将 `isOffDay=false` 且存在 `name` 视为调休补班。

- **🔄 强大的重复规则**
  
  - **按周**：常规周一至周日选择。
  
  - **按月**：每月固定日期（如：每月 10 号发薪日提醒）。
  
  - **按年**：每年固定日期（如：生日提醒）。
  
  - **单次**：精准的单次提醒。

- **🧠 智能补货系统 (Self-Replenishment)**
  
  - 采用 Fixed Date 策略预埋未来闹钟。
  
  - **自动续航**：当用户点击通知栏的“停止”按钮时，通过 AppIntent 自动检测剩余预埋数量。
  
  - **无感更新**：若预埋不足（< 5次），系统会在后台静默重新计算并调度的闹钟。

- **🔔 交互式通知**
  
  - 深度集成 AppIntents。
  
  - 支持在锁屏界面直接操作“停止”或“稍后提醒”，无需进入 App。

- **💾 现代化存储**
  
  - 使用 **SwiftData** 进行本地持久化，数据结构清晰，性能卓越。

- **🎨 优雅的 UI**
  
  - 原生 SwiftUI 打造，流畅的动画与转场。
  
  - 支持深色模式。
